systemd - autorun service for server

1. Creating a service file named <name> and open it in an editor
	sudo nano /etc/systemd/system/<name>.service

2. Content:
[Unit]
Description=<text>
After=network.target

[Service]
User=root
WorkingDirectory=<path/to/working/dir>
ExecStart=<starting comand>
Restart=always

[Install]
WantedBy=multi-user.target
